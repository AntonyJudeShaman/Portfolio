Working

The working involved here is that first instead of barcode tags RFID tags are used. The basket in our prototype is actually RFID scanner which detects the rfid tags placed inside the basket and reads the data in it.
So during purchase while placing the products in the basket it just detects the rfid tags and reads the data that is price of the product and then sends it to the mobile phone.
It contains the applications which is used for tallying up, displaying, generating QR code through NFC based applications and transferring the data about the purchase to some specific apps used for storing the details.
